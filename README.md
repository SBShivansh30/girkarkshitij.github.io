# Portfolio Website

Created with React.js & Tailwind CSS

## Live Application URL

[https://girkarkshitij.github.io/](https://girkarkshitij.github.io/)

## Prerequisites 

Before you can run this React app, please make sure you have [Yarn](https://yarnpkg.com/) installed. If you don't have Yarn installed, you can follow these instructions to install it:

### macOS

Open the terminal and run the following command:

```bash
brew install yarn
```

### Windows

Open a command prompt or PowerShell window and run the following command:

```bash
npm install -g yarn
```

## Installation

Once you have Yarn installed, you can install the necessary dependencies for this app by running the following command:

```bash
yarn install
```

## Usage

Runs the app in the development mode.\
Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

```bash
yarn start
```

## Build

Builds the app for production to the `build` folder.\
It correctly bundles React in production mode and optimizes the build for the best performance.

```bash
yarn build
```
